Вычисление значения выражения:  
{выражение}  
Пример: (3*4-10/2)^2  

Инициализация переменной:  
:i {наименование переменной} = {выражение}  
Пример: :i a = pi*4/2  

Инициализация функции:  
:i {наименование функции}({тип_данных параметр 1}, {тип_данных параметр 2}, ..., {тип_данных параметр n}) = {выражение}  
Пример: :i sum(a, b, c) = a+b+c  

Проверка типа:
:t {наименование переменной}
:t {наименование функции}()
Примеры:
	:t pi
	:t sum()
